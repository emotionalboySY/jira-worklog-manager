최종 수정: 2026-04-17 · Jira 작업 로그 매니저

1. 서비스 개요

본 서비스는 Atlassian Jira Cloud 사용자가 자신의 이슈와 작업 로그를 조회, 기록, 관리할 수 있도록 돕는 개인 도구입니다. 사용자는 자신의 Atlassian 계정으로 OAuth 2.0 인증을 통해 로그인하며, Atlassian에서 허용한 권한 범위 내에서만 본 서비스를 이용할 수 있습니다.

2. 이용 조건

  • 본 서비스는 유효한 Atlassian Jira Cloud 계정을 가진 이용자에게만 제공됩니다.
  • 사용자는 자신의 계정 정보, OAuth 토큰의 관리 책임을 스스로 집니다.
  • 본 서비스는 현재 상태 그대로(AS IS) 제공되며, 특정 목적에 대한 적합성이나 무결성을 보증하지 않습니다. 본 서비스 사용으로 인해 발생한 데이터 손실, 업무 차질에 대해 제공자는 책임지지 않습니다.

3. 금지 행위

  • Atlassian 이용 약관 및 본 서비스의 의도된 사용 범위를 벗어난 행위
  • 다른 사용자의 계정·토큰을 무단으로 획득하거나 본 서비스 코드를 악의적으로 변경하는 행위
  • API를 남용하여 Atlassian 또는 본 서비스의 정상 운영을 방해하는 행위

4. 데이터 처리

본 서비스의 데이터 처리 방식은 개인정보 처리방침을 참고하세요. 사용자의 모든 데이터는 사용자 본인의 브라우저에만 저장되며, 제공자는 사용자 데이터를 별도로 수집·저장하지 않습니다.

5. 서비스 변경 및 종료

본 서비스는 개인이 제공하는 비영리 도구이며, 사전 예고 없이 기능이 변경되거나 서비스가 중단될 수 있습니다. 중단 시 사용자 브라우저에 저장된 데이터는 그대로 유지되지만, Atlassian API 호출 기능은 더 이상 동작하지 않을 수 있습니다.

6. 준거법

본 약관은 대한민국 법령을 준거법으로 하며, 분쟁이 발생할 경우 서울중앙지방법원을 관할 법원으로 합니다.
